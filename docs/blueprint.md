# **App Name**: OlivaTable

## Core Features:

- Dynamic Page Content Display: Render structured sections like Hero, Gastronomic Visual, Our History, Contact, displaying static text, images, and CTA buttons on a single, smooth-scrolling page.
- Responsive Header & Navigation: Implement a sticky header with the restaurant name/logo and a responsive hamburger menu providing smooth-scroll navigation to page sections (Inicio, Cardápio, Reservas, Nossa História, Contato).
- Newsletter Subscription Form: Capture user name and email through a form in the newsletter section and securely store this lead data in a Firebase Firestore collection.
- Interactive Reservation Call-to-Action: Display a prominent 'RESERVE AGORA SUA MESA' button that visually directs users to the primary reservation mechanism (e.g., a simple contact form, external link, or an on-page section if expanded beyond MVP).
- SEO & Metadata Management: Implement Next.js specific metadata for basic SEO, including setting page title and meta description for 'OLIVA MONTADITOS – Reservas & Experiência Gastronômica'.
- Cookie Consent Management: Display a persistent cookie consent banner upon first visit, store user's acceptance or refusal, and control the activation of non-essential scripts (like remarketing placeholders) based on consent.

## Style Guidelines:

- Primary color: A sophisticated olive green (#699954) to reflect the 'Oliva' theme and premium artisanal gastronomy. This hue provides an elegant base, drawing on natural essence.
- Background color: A very dark, almost black, desaturated green-gray (#1F211A) providing a deep, elegant, and contemplative canvas that allows other elements to stand out, aligning with a premium dark theme.
- Accent color: A warm, soft golden hue (#D6B63D) for calls-to-action and key highlights. This color, analogous to the primary green, introduces a touch of luxury and 'dourado suave' without being ostentatious.
- Headlines and prominent titles will use 'Alegreya', an elegant humanist serif font that conveys a sense of intellectual and contemporary luxury, perfect for 'elegante e forte' headings.
- Body text will utilize 'Inter', a versatile grotesque-style sans-serif font known for its legibility and modern neutrality, ensuring 'simples e legível' text throughout the site.
- Utilize minimalist line icons for navigation and informational elements, maintaining the sophisticated and clean aesthetic suitable for a premium brand, especially for the hamburger menu.
- The site will feature a vertical, one-page layout with smooth scrolling between sections, designed with a mobile-first approach to ensure optimal responsiveness and visual hierarchy on all devices.
- Implement subtle fade-in animations on scroll to progressively reveal content, enhancing user engagement, and gentle hover effects on buttons for an interactive and refined user experience.
